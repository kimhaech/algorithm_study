### level-1 최소 직사각형

- code
``` python
def solution(sizes):
    answer = 0
    left = []
    right = []
    
    for size in sizes:
        if size[0] > size[1]:
            left.append(size[1])
            right.append(size[0])
        else:
            left.append(size[0])
            right.append(size[1])
    
    answer = max(left) * max(right)
    
    return answer
```

- 풀이

1. 각 명함의 가로 세로를 비교하여 큰 값은 right, 작은 값은 left에 넣는다.
2. left, right의 최대값을 곱한다.

    class Solution {
        public int solution(int[][] sizes) {
            int answer = 0;
            int max_l = 1;
            int max_s = 1;
            for (int i = 0; i < sizes.length; i++) {
                if (sizes[i][0] > sizes[i][1]) {
                    if (max_l < sizes[i][0])
                        max_l = sizes[i][0];
                    if (max_s < sizes[i][1])
                        max_s = sizes[i][1];
                } else{
                    if (max_l < sizes[i][1])
                        max_l = sizes[i][1];
                    if (max_s < sizes[i][0])
                        max_s = sizes[i][0];
                }   
            }
            
            answer = max_l * max_s;
            
            return answer;
        }
    }
    
    // 각 명함의 가로, 세로 중 큰 길이를 1번 배열에, 작은 길이를 2번 배열에 넣는다.
    // 1번 배열과 2번 배열의 각 최댓값을 구하고 해당 값들을 곱한 것이 answer가 된다.
    // 위의 방법에서는 정렬을 다시 하기 때문에 수행시간이 늘어난다
    // 매 반복에서 원소를 비교하고, smaller, lager 배열을 비교하고 각 배열의 최대값을 찾는 과정을 함께한다.
