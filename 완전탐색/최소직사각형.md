### level-1 최소 직사각형

- code
``` python
def solution(sizes):
    answer = 0
    left = []
    right = []
    
    for size in sizes:
        if size[0] > size[1]:
            left.append(size[1])
            right.append(size[0])
        else:
            left.append(size[0])
            right.append(size[1])
    
    answer = max(left) * max(right)
    
    return answer
```

- 풀이

1. 각 명함의 가로 세로를 비교하여 큰 값은 right, 작은 값은 left에 넣는다.
2. left, right의 최대값을 곱한다.
